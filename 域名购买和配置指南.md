## 🚀 配置Vercel自定义域名

### 步骤1：在Vercel添加域名

1. **登录Vercel**
   - 访问：https://vercel.com
   - 进入您的项目：`seo-optimizer-pro`

2. **进入域名设置**
   - 点击项目设置
   - 选择 **"Domains"** 标签页
   - 点击 **"Add"** 或 **"Add Domain"**

3. **输入域名**
   - 输入您购买的域名（如：`seooptimizer.com`）
   - 点击 **"Add"**

4. **选择配置方式**
   - Vercel会显示两种配置方式：
     - **A Record**（推荐）
     - **CNAME Record**

## 📝 DNS配置

### 方法1：使用CNAME（推荐，最简单）

#### 在域名注册商配置DNS

1. **登录域名注册商**（Namecheap/GoDaddy等）

2. **进入DNS管理**
   - Namecheap: Domain List → Manage → Advanced DNS
   - GoDaddy: My Products → DNS

3. **添加CNAME记录**
   
   **如果使用根域名**（如：`seooptimizer.com`）：
   - 类型：**CNAME**
   - 主机记录：**@** 或留空
   - 记录值：**cname.vercel-dns.com**
   - TTL：自动或3600

   **如果使用子域名**（如：`www.seooptimizer.com`）：
   - 类型：**CNAME**
   - 主机记录：**www**
   - 记录值：**cname.vercel-dns.com**
   - TTL：自动或3600

4. **保存设置**

### 方法2：使用A记录

如果域名注册商不支持根域名的CNAME，使用A记录：

1. **在Vercel获取IP地址**
   - Vercel会显示需要配置的A记录IP地址
   - 通常是：`76.76.21.21`

2. **在域名注册商添加A记录**
   - 类型：**A**
   - 主机记录：**@** 或留空
   - 记录值：**76.76.21.21**（Vercel提供的IP）
   - TTL：自动或3600

3. **同时添加www子域名**（CNAME）
   - 类型：**CNAME**
   - 主机记录：**www**
   - 记录值：**cname.vercel-dns.com**

## ⏳ 等待DNS生效

### DNS传播时间

- **通常**：5分钟 - 2小时
- **最长**：24-48小时

使用在线工具检查：
- https://dnschecker.org
- https://www.whatsmydns.net

输入您的域名，检查是否指向Vercel的IP。

## 🔧 配置Render自定义域名（后端，可选）

### 步骤1：在Render添加域名

1. **登录Render**
   - 访问：https://render.com
   - 进入您的Web Service

2. **进入设置**
   - 点击 **"Settings"** 标签页
   - 找到 **"Custom Domains"** 部分

3. **添加域名**
   - 点击 **"Add Custom Domain"**
   - 输入子域名（如：`api.seooptimizer.com`）
   - 点击 **"Save"**

4. **获取DNS配置**
   - Render会显示需要配置的DNS记录
   - 通常是CNAME记录

### 步骤2：配置DNS

在域名注册商添加CNAME记录：

- 类型：**CNAME**
- 主机记录：**api**（或您想要的子域名）
- 记录值：Render提供的值（如：`xxx.onrender.com`）
- TTL：自动或3600

## 🔄 更新配置

### 更新前端配置

如果后端也使用了自定义域名，更新 `config.js`：

```javascript
production: {
    apiBaseUrl: 'https://api.seooptimizer.com/api'
    // 或继续使用Render域名
    // apiBaseUrl: 'https://seo-optimizer-pro.onrender.com/api'
}
```

### 更新CORS配置

如果后端使用自定义域名，更新Render的CORS_ORIGIN：

```
CORS_ORIGIN=https://seooptimizer.com
```

## 🐛 常见问题

### 问题1：DNS未生效

**解决**：
- 等待更长时间（最长48小时）
- 清除DNS缓存
- 检查DNS记录是否正确

### 问题2：SSL证书未配置

**解决**：
- 等待Vercel/Render自动配置（通常几分钟）
- 检查DNS是否正确指向
- 确认域名已验证

### 问题3：网站无法访问

**解决**：
- 检查DNS记录是否正确
- 确认域名已添加到Vercel
- 检查浏览器控制台错误
